# Loom Script — Dataleap Quickstart Hub Demo

**Target length: 3-4 minutes**
**Tone: Confident, casual, energetic — like you're talking to a co-founder, not pitching a VC**

---

## Opening (15 seconds)

"Hey Jan — I'm Niran. I'm applying for the Chief of Staff role, and instead of just sending a resume, I wanted to show you what I'd actually do on the job. So this weekend, I built three things."

## The Problem (30 seconds)

"I signed up for Dataleap and the first thing I noticed is that new users face a blank canvas problem — they know AI agents are powerful, but they don't know what to automate first. This is the same activation challenge every flexible tool faces. Notion solved it with templates. Zapier solved it with their template directory, which now drives 8.5 million monthly visitors. No AI agent platform has built this yet."

## Demo: Quickstart Hub (60 seconds)

*Screen share the deployed Vercel app*

"So I built this — the Dataleap Quickstart Hub. It's 25 curated agent templates organized by role."

*Click through filters*

"You can filter by Sales, Ops, Engineering — each card shows time saved, setup time, and the tools required."

*Click into a template detail*

"The key differentiator is this — every template has complete, copy-paste-ready agent instructions. A new user can literally grab these, paste them into Dataleap, and have a working agent in minutes."

*Navigate to Advisor*

"And for users who don't know where to start — there's an AI advisor. It asks about your role and pain points, then recommends specific templates. This is a genuine whitespace play — no competitor has this."

## Demo: Dataleap Agents (45 seconds)

*Show Dataleap dashboard with your agents*

"I also built three agents directly on the platform. This is a competitor intel digest that monitors Lindy, Relevance, and Bardeen weekly."

*Show an agent run output*

"Here's the actual output — it caught [mention something specific from the run]. I also built an onboarding agent that could welcome new signups, and a content repurposer. Building these took me about 20 minutes each — which is exactly the story we want every new user to experience."

## Strategy (45 seconds)

"All of this connects to a bigger growth thesis. Dataleap's number one lever right now is reducing time to first agent. The template gallery drives organic discovery through SEO — the same playbook that built Zapier's top of funnel. The AI advisor converts curious visitors into activated users. And the onboarding agent uses your own product to solve the activation problem."

"I laid out a full 6-month playbook in a strategy memo — I'll link it below."

## Close (15 seconds)

"I built all of this in a weekend because I wanted to show you how I work — fast, autonomous, and focused on what actually moves the needle. I'd love to do this full-time. Looking forward to chatting."

---

## Tips for Recording

- Share your screen with the Vercel app open in a browser tab, Dataleap open in another
- Use a clean desktop, no clutter
- Speak at a natural pace — don't rush
- Smile when you say the opener and closer
- If you mess up, just keep going — scrappy > polished for a startup
- Record in one take if possible — it shows confidence
